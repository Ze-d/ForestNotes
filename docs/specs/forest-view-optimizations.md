# Forest View 3D 优化清单

> 当前状态: Three.js 3D 等距悬浮岛基础框架已就绪，以下为待优化项。

---

## 1. 缩放交互

- [ ] **鼠标滚轮缩放** — 滚动缩放 OrthographicCamera 的 viewSize，范围建议 6–20
- [ ] **平滑缩放动画** — 缩放时加入缓动，避免突兀跳变
- [ ] **缩放到选中树** — 点击树时自动 zoom in 到该树附近
- [ ] **双指缩放**（触控板支持）— `wheel` 事件的 `ctrlKey` 检测

## 2. 树木 & 藤蔓美术优化

### 树木

- [ ] **树干质感** — 添加纹理或噪声法线贴图，模拟树皮（低多边形风格保持一致）
- [ ] **树冠形态多样性** — 不同树用不同的几何体组合（锥形/球形/椭球形），避免所有树一模一样
- [ ] **树冠颜色渐变** — 同一棵树的多层树冠用同色系深浅变化，更有层次感
- [ ] **季节性/健康度视觉** — healthy 翠绿饱满 → stale 黄绿稀疏 → dormant 枯灰光秃
- [ ] **树叶粒子效果** — healthy 树周围漂浮小粒子（落叶/光点）
- [ ] **树的排布算法** — 优化 `assignGridPositions`：
  - 大 tag 树（noteCount 高）放在岛中心位置
  - 小 tag 树放在边缘
  - 同 cluster（藤蔓连接多）的树聚集在一起
  - 树之间最小间距保证不穿插
- [ ] **树的生长动画** — 初次加载时树从地面冒出（scale 0→1, 0.5s 缓动）

### 藤蔓

- [ ] **藤蔓形态** — 当前为简单贝塞尔曲线，改用 TubeGeometry 增加粗细感
- [ ] **藤蔓缠绕效果** — 两棵树之间的藤蔓沿树干螺旋上升再连接
- [ ] **藤蔓颜色** — 根据两端树的健康度取中间色
- [ ] **藤蔓密度** — 共现次数 weight 高时画多条藤蔓（weight 条）

## 3. 整体场景美术

### 悬浮岛

- [ ] **岛屿形状** — 当前为方形块，改为不规则形状（圆形/椭圆/自然切割面）
- [ ] **泥土截面纹理** — 添加岩石纹理、层次分明的土层颜色（表层深棕 → 中层浅棕 → 底层灰石）
- [ ] **草地细节** — 表面添加低多边形小花、小草丛、小蘑菇等装饰物
- [ ] **岛屿边缘** — 泥土截面添加碎石/凸起几何体
- [ ] **岛屿底面** — 悬浮岛底部添加垂下的树根/藤蔓/钟乳石状泥土

### 背景

- [ ] **天空盒/背景渐变** — 当前为纯色 `#1a1b2e`，改为渐变（深蓝紫 → 柔和蓝灰）
- [ ] **星空/云层** — 背景添加静态低多边形云朵或星点粒子
- [ ] **远景元素** — 远处添加虚拟的小型悬浮岛（silhouette），增加景深感

### 光照

- [ ] **柔和阴影** — 当前阴影偏硬，调整 shadow map 参数或使用 softer shadows
- [ ] **环境光遮蔽 (AO)** — 树木根部与草地接触处加暗
- [ ] **岛下投影** — 悬浮岛在下方投射柔和阴影（如在地面上）

### 氛围

- [ ] **治愈系色调** — 整体 post-processing：柔和暖色调
- [ ] **雾效优化** — 当前雾色与背景色一致，调整雾的密度让远景元素淡出
- [ ] **粒子效果** — 萤火虫光点、飘浮花粉、落叶

---

## 优先级建议

| 优先级 | 类别 | 说明 |
|--------|------|------|
| P0 | 滚轮缩放 | 基础交互缺失 |
| P0 | 树冠多样性 + 健康色 | 当前所有树一模一样，无法区分 |
| P1 | 树排布优化 | 大 tag 应更突出 |
| P1 | 藤蔓形态 (TubeGeometry) | 当前线条太细 |
| P2 | 岛屿形状 + 材质 | 方形块不够自然 |
| P2 | 天空/背景 | 纯色背景单调 |
| P3 | 粒子/动画 | 锦上添花 |

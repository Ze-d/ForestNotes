# Note-taking Tools Comparison

A quick comparison of popular #note-taking tools for #knowledge-management.

## Tools

### Obsidian

- Local-first #markdown files ✅
- Graph view
- Plugin ecosystem
- Proprietary (not open source)

### Logseq

- Open source
- Local-first #markdown
- Outliner-based
- Block references

### Notion

- Cloud-first (data not local)
- Rich block editor
- Collaboration features
- Vendor lock-in risk

### ForestNotes 🌳

- Local-first #markdown — your files, your disk
- Knowledge Forest visualization
- SQLite FTS5 full-text search
- Open source
- Tauri 2 desktop app

## Why Local-First Matters

| Feature | Cloud Tools | Local-First |
|---------|-------------|-------------|
| Offline access | Limited | ✅ Full |
| Data ownership | Platform owns | ✅ You own |
| Portability | Export needed | ✅ Plain .md |
| Privacy | Server-side | ✅ Local only |
| Longevity | Platform risk | ✅ Files last forever |

For serious #productivity and long-term #knowledge-management, local-first #markdown tools are the safest investment.

# ForestNotes Sample Vault

This folder is a demo vault for ForestNotes. Open it in the app to see the Knowledge Forest in action.

## How to Use

1. Launch ForestNotes
2. Click **Open** in the sidebar
3. Select this `sample-vault/` folder
4. Switch to **Forest** view to see the Knowledge Forest
5. Click trees to explore tags and their connections

## Expected Forest View

With these notes, you should see:

- **AI Cluster**: AI ↔ Transformer ↔ LLM ↔ deep-learning ↔ RAG ↔ prompt-engineering — tightly connected
- **Frontend Cluster**: typescript ↔ react ↔ web-development ↔ programming
- **Productivity Cluster**: productivity ↔ knowledge-management ↔ note-taking
- **Bridge Tags**: AI+productivity via prompt-engineering; note-taking+markdown connects productivity to the broader ecosystem

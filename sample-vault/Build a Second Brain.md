# Building a Second Brain

The #note-taking methodology popularized by Tiago Forte emphasizes capturing, organizing, and retrieving knowledge.

## The CODE Framework

- **C**apture: Save what resonates
- **O**rganize: Structure for actionability
- **D**istill: Find the essence
- **E**xpress: Share your work

## PARA Method

Organize notes into four categories:

1. **P**rojects: Active tasks with deadlines
2. **A**reas: Ongoing responsibilities
3. **R**esources: Reference materials
4. **A**rchives: Completed or inactive items

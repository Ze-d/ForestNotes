悬浮岛视觉重设计

P2：当前方形块岛屿 → 更自然的有机形态

当前 vs 目标对比

当前：方块岛屿

BoxGeometry 堆叠、直角边缘

目标：有机形态岛屿

不规则椭圆截面 + 分层泥土 + 曲面草地

岛屿改进方案

1

不规则椭圆截面（推荐）

用 ExtrudeGeometry + 自定义 Shape 替换 BoxGeometry 堆叠。
形状：椭圆 + 2-3 层 Perlin/正弦扰动 → 自然切割面

2

泥土分层纹理

截面三层：表层深棕 → 中层浅棕 → 底层灰石。
用 MeshStandardMaterial + roughness map 替代纯色

3

边缘细节：碎石 + 草根

岛屿侧面随机散布小碎石（icosahedron）和垂下的草根（细圆柱体）
打破光滑边缘，增加自然感

4

岛底悬垂物

底部垂下树根、藤蔓、钟乳石状泥土
用随机锥形体沿底面分布

草地装饰细节

5

草丛 / 小花

岛面散布小草丛（ConeGeometry 矮圆锥 + 随机颜色小花瓣）
每棵树下 3-5 簇，颜色随机

6

小蘑菇装饰

树根部偶尔放置低多边形蘑菇（cap + stem）
治愈系点睛，不需要太多

推荐优先级

1. 椭圆截面必做 方形→圆形是最直观的改进
2. 泥土分层必做 用不同材质颜色即可，成本低
3. 边缘碎石推荐 InstancedMesh 几十个小石头
5. 草丛小花推荐 InstancedMesh，数量可控
4. 岛底悬垂可选 正常视角看不太到
6. 小蘑菇可选 锦上添花
